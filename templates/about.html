About This Project

This AI-based poultry disease detection system uses a deep learning model that is used to classify images of chicken droppings into four categories: Coccidiosis, Newcastle Disease, Salmonella, and Healthy. The project aims to assist farmers with quick, on-the-spot diagnosis using image input. It helps reduce the economic impact of delayed treatment and improves poultry health management efficiency. The model was trained using transfer learning with over so many images and tested for reliable accuracy. This web interface is user-friendly and mobile-accessible.
